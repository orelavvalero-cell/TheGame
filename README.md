# The Game

2D multiplayer tank prototype for Android.

## What is inside

- Huge 6200x6200 world; the camera follows your tank.
- Two-player Wi-Fi mode:
  - Device 1: press `HOST`.
  - Device 2: press `FIND`.
  - Both devices must be on the same Wi-Fi network.
- 16 AI tanks with chasing, strafing, line-of-fire checks and shooting.
- Touch controls:
  - Left pad: move.
  - Right pad: aim and fire.
- Native Android Java code, no external game engine.

## Open in Android Studio

1. Open this folder: `C:\Users\Валерий\TheGame`.
2. Let Android Studio sync Gradle.
3. Run on a phone or emulator.

For real Wi-Fi multiplayer, install the app on two Android devices connected to the same router. Some public/hotel networks block device-to-device traffic; a home router or phone hotspot usually works better.
